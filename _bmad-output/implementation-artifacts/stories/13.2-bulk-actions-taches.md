# Story 13.2 : Bulk actions completes sur les taches

## User Story

As a **colocataire**,
I want **pouvoir selectionner plusieurs taches et les supprimer ou les marquer "Terminee" en une action**,
So that **je gagne du temps dans la gestion quotidienne**.

## Description

Actuellement, la barre de bulk actions dans `Tasks.jsx` (L279-298) ne propose que la delegation ("Deleguer a..."). Il manque deux actions essentielles : "Marquer Terminee" (changement de statut en masse) et "Supprimer" (suppression en masse avec confirmation). La barre flottante apparait deja lorsque `selectedTasks.length > 0`, il suffit d'y ajouter les deux boutons.

## Criteres d'Acceptation

**Given** j'ai selectionne 3 taches via les checkboxes
**When** je clique "Marquer Terminee" dans la barre d'actions groupees
**Then** les 3 taches passent en statut "Terminee" et se deplacent dans la colonne correspondante
**And** la selection est reinitialise

**Given** j'ai selectionne 2 taches
**When** je clique "Supprimer" dans la barre d'actions groupees
**Then** une popup de confirmation s'affiche
**And** apres confirmation, les 2 taches sont supprimees
**And** la selection est reinitialise

**Given** j'ai selectionne des taches et je clique "Supprimer"
**When** je clique "Annuler" dans la popup de confirmation
**Then** rien n'est supprime et la selection est conservee

**Given** la barre de bulk actions est visible
**When** je regarde les actions disponibles
**Then** je vois : "Deleguer a...", "Marquer Terminee", "Supprimer", "Annuler"

## Taches Techniques

### Task 1 : Ajouter la fonction `bulkMarkDone` dans Tasks.jsx

Ajouter apres la fonction `bulkReassign` (L149-164) :

```javascript
async function bulkMarkDone() {
  try {
    const updates = await Promise.all(
      selectedTasks.map((id) => updateTask(id, { status: "Terminee" }))
    )
    setTasks((prev) =>
      prev.map((t) => {
        const upd = updates.find((u) => u.id === t.id)
        return upd || t
      })
    )
    setSelectedTasks([])
  } catch (err) {
    console.error("Bulk mark done error:", err)
  }
}
```

### Task 2 : Ajouter la fonction `bulkDelete` dans Tasks.jsx

Ajouter apres `bulkMarkDone` :

```javascript
const [bulkDeleteConfirmOpen, setBulkDeleteConfirmOpen] = useState(false)
const [isBulkDeleting, setIsBulkDeleting] = useState(false)

async function confirmBulkDelete() {
  setIsBulkDeleting(true)
  try {
    await Promise.all(selectedTasks.map((id) => deleteTask(id)))
    setTasks((prev) => prev.filter((t) => !selectedTasks.includes(t.id)))
    setSelectedTasks([])
  } catch (err) {
    console.error("Bulk delete error:", err)
  } finally {
    setIsBulkDeleting(false)
    setBulkDeleteConfirmOpen(false)
  }
}
```

N'oublier pas d'ajouter les state `bulkDeleteConfirmOpen` et `isBulkDeleting` dans les declarations de state en haut du composant (apres L46).
