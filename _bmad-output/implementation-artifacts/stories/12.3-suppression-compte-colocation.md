# Story 12.3 : Suppression de compte et de colocation

## Description

Le bouton "Supprimer le compte" existe dans Settings.jsx (ligne 299) mais n'a aucun handler `onClick`. Il n'y a pas non plus de bouton pour supprimer une colocation entiere. Cette story implemente les deux fonctionnalites avec les routes API correspondantes, les confirmations de securite, et le nettoyage complet des donnees.

## User Story

As a **utilisateur**,
I want **pouvoir supprimer mon compte ou, en tant qu'admin, supprimer toute la colocation**,
So that **je peux quitter definitivement l'application ou dissoudre la colocation**.

## Criteres d'Acceptation

**AC-1: Suppression de compte (membre)**
**Given** je suis membre (non-admin) et je clique "Supprimer le compte"
**When** la popup de confirmation s'affiche et je confirme
**Then** mon compte est supprime de la liste des utilisateurs
**And** je suis retire de la colocation (`members` mis a jour)
**And** `sessionStorage` est nettoye et je suis redirige vers `/login`

**AC-2: Suppression de compte (admin, pas le seul)**
**Given** je suis admin et il y a au moins un autre admin
**When** je clique "Supprimer le compte" et confirme
**Then** mon compte est supprime et je suis retire de la colocation
**And** la colocation continue d'exister avec les autres membres

**AC-3: Suppression de compte (seul admin)**
**Given** je suis le seul admin de la colocation
**When** je clique "Supprimer le compte"
**Then** un message m'informe "Vous etes le seul admin. Promouvez un autre membre avant de supprimer votre compte."
**And** mon compte n'est PAS supprime

**AC-4: Bouton "Supprimer la colocation" visible pour admin uniquement**
**Given** je suis admin sur la page Reglages
**When** je consulte le bas de la page
**Then** un bouton rouge "Supprimer la colocation" est visible
**And** si je suis membre (non-admin), ce bouton n'est PAS visible

**AC-5: Suppression de la colocation avec double confirmation**
**Given** je suis admin et je clique "Supprimer la colocation"
**When** la popup de confirmation s'affiche avec un champ de re-saisie du nom de la colocation
**Then** le bouton "Confirmer la suppression" est desactive tant que le nom saisi ne correspond pas
**And** apres confirmation, la colocation et toutes ses donnees associees sont supprimees

**AC-6: Nettoyage complet apres suppression de colocation**
**Given** la colocation est supprimee
**When** le serveur traite la requete
**Then** toutes les taches, finances, abonnements, recettes et articles de courses lies sont supprimes
**And** tous les membres ont leur `colocationId` mis a null
**And** tous les membres sont deconnectes (cote client: `sessionStorage` nettoye, redirection `/login`)

## Taches Techniques

### Task 1: Creer la route DELETE /api/users/:id

**File:** `server/index.js`

Ajouter apres la route `PUT /api/users/:id` (ligne ~113):

```javascript
app.delete('/api/users/:id', (req, res) => {
  const idx = users.findIndex(u => u.id === req.params.id);
  if (idx === -1) return res.status(404).json({ error: 'Utilisateur non trouve' });

  const targetUser = users[idx];

  // Check if user is the last admin
  if (targetUser.role === 'admin' && targetUser.colocationId) {
    const colocMembers = users.filter(u =>
      colocation.members.includes(u.id) && u.role === 'admin'
    );
    if (colocMembers.length <= 1) {
      return res.status(400).json({
        error: 'Vous etes le seul admin. Promouvez un autre membre avant de supprimer votre compte.'
      });
    }
  }

  // Remove user from colocation members
  const memberIdx = colocation.members.indexOf(req.params.id);
  if (memberIdx !== -1) {
    colocation.members.splice(memberIdx, 1);
    db.save('colocation', colocation);
  }

  // Remove user from users array
  users.splice(idx, 1);
  db.save('users', users);

  res.json({ data: { message: 'Compte supprime' } });
});
```

### Task 2: Creer la route DELETE /api/colocation/:id

**File:** `server/index.js`

Ajouter apres la route `POST /api/colocation/join` (ou apres les routes members):

```javascript
app.delete('/api/colocation/:id', (req, res) => {
  if (colocation.id !== req.params.id) {
    return res.status(404).json({ error: 'Colocation non trouvee' });
  }

  // Clear all colocation-linked data
  tasks.length = 0;
  finances.length = 0;
  subscriptions.length = 0;
  recipes.length = 0;
  shoppingList.length = 0;

  db.save('tasks', tasks);
  db.save('finances', finances);
  db.save('subscriptions', subscriptions);
  db.save('recipes', recipes);
  db.save('shoppingList', shoppingList);

  // Set all members' colocationId to null
  colocation.members.forEach(memberId => {
    const u = users.find(usr => usr.id === memberId);
    if (u) u.colocationId = null;
  });
  db.save('users', users);

  // Reset colocation object
  colocation.members = [];
  colocation.name = '';
  colocation.invitationCode = '';
  colocation.totalFund = 0;
  db.save('colocation', colocation);

  res.json({ data: { message: 'Colocation supprimee' } });
});
```
