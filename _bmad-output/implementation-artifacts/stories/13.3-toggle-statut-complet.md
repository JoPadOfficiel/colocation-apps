# Story 13.3 : Toggle statut complet (A faire / En cours / Terminee)

## User Story

As a **colocataire**,
I want **pouvoir faire passer une tache par les 3 statuts : A faire -> En cours -> Terminee**,
So that **je peux suivre precisement l'avancement de chaque tache**.

## Description

Actuellement, la fonction `toggleStatus` dans `Tasks.jsx` (L139-147) ne gere que 2 etats : elle fait basculer entre "Terminee" et "A faire", en ignorant completement le statut "En cours". De plus, l'affichage Kanban ne montre que 2 colonnes ("A Faire" et "Terminees") alors qu'il devrait y en avoir 3. Le composant `TaskCard.jsx` ne montre que 2 etats visuels (done/not done) sans differencier "A faire" de "En cours".

## Criteres d'Acceptation

**Given** une tache est "A faire"
**When** je clique sur le bouton statut (cercle)
**Then** elle passe a "En cours"

**Given** une tache est "En cours"
**When** je clique sur le bouton statut
**Then** elle passe a "Terminee"

**Given** une tache est "Terminee"
**When** je clique sur le bouton statut
**Then** elle repasse a "A faire"

**Given** je suis sur la page Taches
**When** je regarde les colonnes
**Then** je vois 3 colonnes : "A Faire", "En cours", "Terminees"

**Given** une tache est "En cours"
**When** je regarde sa TaskCard
**Then** le cercle de statut est visuellement different de "A faire" et "Terminee" (ex: demi-rempli, couleur bleu/orange)

## Taches Techniques

### Task 1 : Corriger `toggleStatus` dans Tasks.jsx

Modifier la fonction `toggleStatus` a la ligne 139-147 :

```javascript
// BEFORE (L139-147):
async function toggleStatus(task) {
  const newStatus = task.status === "Terminee" ? "A faire" : "Terminee"
  // ...
}

// AFTER:
async function toggleStatus(task) {
  const statusCycle = {
    "A faire": "En cours",
    "En cours": "Terminee",
    "Terminee": "A faire",
  }
  const newStatus = statusCycle[task.status] || "A faire"
  try {
    const updated = await updateTask(task.id, { status: newStatus })
    setTasks((prev) => prev.map((t) => (t.id === task.id ? updated : t)))
  } catch (err) {
    console.error("Status toggle error:", err)
  }
}
```

### Task 4 : Mettre a jour les statistiques pour inclure "En cours"

Dans `Tasks.jsx` L211-219, ajouter le comptage "En cours" :

```javascript
// BEFORE:
users.forEach((u) => { stats[u.id] = { total: 0, done: 0 } })

// AFTER:
users.forEach((u) => { stats[u.id] = { total: 0, done: 0, inProgress: 0 } })
tasks.forEach((t) => {
  if (stats[t.assignedTo]) {
    stats[t.assignedTo].total++
    if (t.status === "Terminee") stats[t.assignedTo].done++
    if (t.status === "En cours") stats[t.assignedTo].inProgress++
  }
})
```
