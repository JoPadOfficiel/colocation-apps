# Story 11.1 : Alimenter la cagnotte commune

## User Story

As a **colocataire**,
I want **pouvoir ajouter de l'argent a la cagnotte commune**,
So that **la cagnotte reflete les contributions reelles de chaque membre**.

## Description

Actuellement, la cagnotte commune (`totalFund`) est un nombre statique defini dans `colocation.totalFund` (245.50 par defaut dans mockData). Il n'existe aucun mecanisme pour ajouter de l'argent a la cagnotte depuis l'interface. La card "Cagnotte Commune" dans `Finances.jsx` affiche le montant mais ne permet aucune interaction.

Cette story ajoute un bouton "Ajouter a la cagnotte" sur la page Finances qui ouvre un Dialog avec un champ montant. La contribution est tracee via une entree `POST /api/finances` avec `type: 'contribution'` et le `totalFund` de la colocation est incremente via `PUT /api/colocation/:id`.

## Criteres d'Acceptation

**Given** je suis sur la page Finances
**When** je clique "Ajouter a la cagnotte" et saisis un montant (ex: 50 EUR)
**Then** la cagnotte commune augmente de 50 EUR
**And** une entree est ajoutee dans l'historique : "Contribution cagnotte -- 50 EUR -- [mon nom] -- [date]"
**And** la card "Cagnotte Commune" se met a jour immediatement

**Given** je tente d'ajouter 0 EUR ou un montant negatif
**When** je valide
**Then** le formulaire refuse la saisie avec un message d'erreur

**Given** j'ajoute une contribution a la cagnotte
**When** je consulte le tableau des depenses
**Then** la contribution apparait avec un badge "Contribution" distinct des depenses

### Task 1 : Ajouter la route PUT /api/colocation/:id pour mettre a jour totalFund

**Fichier : `server/index.js`**

Ajouter une nouvelle route apres la route `POST /api/colocation/join` (line 147) :

```javascript
// Update colocation (totalFund, etc.)
app.put('/api/colocation/:id', (req, res) => {
  if (colocation.id !== req.params.id) {
    return res.status(404).json({ error: 'Colocation non trouvee' });
  }
  const { totalFund, ...otherFields } = req.body;
  if (totalFund !== undefined) {
    if (typeof totalFund !== 'number' || totalFund < 0) {
      return res.status(400).json({ error: 'Montant invalide' });
    }
    colocation.totalFund = totalFund;
  }
  Object.assign(colocation, otherFields);
  db.save('colocation', colocation);
  res.json({ data: enrichColocation(colocation) });
});
```

3. Ajouter le handler de soumission de contribution :
```javascript
const handleCagnotteSubmit = async (e) => {
  e.preventDefault();
  const amount = parseFloat(cagnotteAmount);
  if (!cagnotteAmount || isNaN(amount) || amount <= 0) {
    setCagnotteError("Le montant doit etre superieur a 0");
    return;
  }

  setIsSubmitting(true);
  try {
    // 1. Increment totalFund on colocation
    const newTotal = (colocation.totalFund || 0) + amount;
    await updateColocation(colocation.id, { totalFund: newTotal });

    // 2. Create a finance entry to trace the contribution
    await createFinance({
      title: `Contribution cagnotte`,
      amount: amount,
      date: new Date().toISOString().split('T')[0],
      paidBy: user.id,
      type: 'contribution',
      colocationId: colocation.id
    });

    // 3. Refetch and recalculate
    const data = await fetchFinances();
    setFinances(data);
    setCagnotte(newTotal);
    calculateMetrics(data);

    // 4. Close dialog
    setIsCagnotteDialogOpen(false);
    setCagnotteAmount("");
    setCagnotteError("");
  } catch (error) {
    console.error("Erreur contribution cagnotte:", error);
    alert("Erreur lors de l'ajout a la cagnotte");
  } finally {
    setIsSubmitting(false);
  }
};
```
