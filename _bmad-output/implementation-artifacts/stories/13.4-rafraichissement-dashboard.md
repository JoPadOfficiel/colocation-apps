# Story 13.4 : Rafraichissement du Dashboard et activites recentes

## User Story

As a **colocataire**,
I want **que le dashboard affiche des donnees a jour et que le flux d'activites recentes soit pertinent**,
So that **je vois l'etat reel de ma colocation en temps reel**.

## Description

Le Dashboard (`client/src/pages/Dashboard.jsx`) a 3 problemes :

1. **Donnees stalles** : Le `useEffect` (L183-189) ne se re-execute pas quand on navigue vers le dashboard depuis une autre page (React Router garde le composant monte). Les widgets affichent des donnees obsoletes apres ajout/modification de depenses ou taches.

2. **Activites recentes mal datees** : Le composant `RecentActivity` (L89-137) utilise `t.dueDate` (L103) pour les taches terminees au lieu de `t.updatedAt` ou `t.createdAt`. Une tache terminee aujourd'hui avec une dueDate dans 2 semaines affichera une date future.

3. **Graphique par categorie au lieu de par mois** : Le composant `ExpenseChart` (L37-78) groupe par `f.type` (categorie) et non par mois. Le graphique devrait montrer l'historique multi-mois.

## Criteres d'Acceptation

**Given** j'ajoute une depense depuis la page Finances
**When** je reviens sur le Dashboard (via sidebar ou navigation)
**Then** le widget Finances affiche le nouveau solde
**And** l'activite recente inclut ma depense avec le bon timestamp

**Given** je change le statut d'une tache en "Terminee"
**When** je reviens sur le Dashboard
**Then** le widget Taches affiche le nombre correct de taches urgentes
**And** l'activite recente inclut la tache terminee

**Given** les donnees couvrent plusieurs mois
**When** je consulte le graphique du dashboard
**Then** le graphique affiche l'historique multi-mois (pas uniquement par categorie)

**Given** une tache a ete terminee aujourd'hui
**When** je regarde les activites recentes
**Then** le timestamp affiche "a l'instant" ou "il y a Xh" (pas la date d'echeance)

## Taches Techniques

### Task 1 : Forcer le refetch a chaque navigation vers le Dashboard

Le `useEffect` dans `Dashboard.jsx` (L183-189) a un tableau de dependances vide `[]`. Il ne se re-execute que lors du montage initial. Pour forcer un refetch a chaque visite, utiliser un identifiant de navigation.

Modifier `Dashboard.jsx` L183-189 :

```javascript
// BEFORE:
useEffect(() => {
  Promise.all([fetchTasks(), fetchFinances(), fetchShoppingList(), fetchSubscriptions(), fetchUsers()])
    .then(([tasks, finances, shopping, subscriptions, users]) => {
      setData({ tasks, finances, shopping, subscriptions, users })
    })
    .catch((err) => console.error("Dashboard load error:", err))
    .finally(() => setLoading(false))
}, [])

// AFTER:
import { useLocation } from "react-router-dom"
// ... in component:
const location = useLocation()

useEffect(() => {
  setLoading(true)
  Promise.all([fetchTasks(), fetchFinances(), fetchShoppingList(), fetchSubscriptions(), fetchUsers()])
    .then(([tasks, finances, shopping, subscriptions, users]) => {
      setData({ tasks, finances, shopping, subscriptions, users })
    })
    .catch((err) => console.error("Dashboard load error:", err))
    .finally(() => setLoading(false))
}, [location.key])
```

La `location.key` change a chaque navigation, meme vers la meme route.

### Task 4 : Transformer le graphique en historique multi-mois

Modifier le composant `ExpenseChart` dans `Dashboard.jsx` (L37-78) pour grouper par mois au lieu de par type :

```jsx
function ExpenseChart({ finances }) {
  // Group by month
  const byMonth = {}
  finances.forEach((f) => {
    const d = new Date(f.date)
    const key = `${d.getFullYear()}-${String(d.getMonth() + 1).padStart(2, '0')}`
    byMonth[key] = (byMonth[key] || 0) + f.amount
  })

  const entries = Object.entries(byMonth).sort((a, b) => a[0].localeCompare(b[0]))
  const max = Math.max(...entries.map(([, v]) => v), 1)

  const monthLabels = {
    '01': 'Janvier', '02': 'Fevrier', '03': 'Mars', '04': 'Avril',
    '05': 'Mai', '06': 'Juin', '07': 'Juillet', '08': 'Aout',
    '09': 'Septembre', '10': 'Octobre', '11': 'Novembre', '12': 'Decembre',
  }

  return (
    <Card>
      <CardHeader>
        <CardTitle className="flex items-center gap-2 text-lg">
          <TrendingUp className="w-5 h-5 text-blue-600" />
          Depenses mensuelles
        </CardTitle>
      </CardHeader>
      <CardContent>
        {entries.length === 0 ? (
          <p className="text-sm text-gray-500">Aucune depense enregistree</p>
        ) : (
          <div className="space-y-3">
            {entries.map(([month, amount]) => {
              const [year, m] = month.split('-')
              const label = `${monthLabels[m] || m} ${year}`
              return (
                <div key={month} className="space-y-1">
                  <div className="flex justify-between text-sm">
                    <span className="text-gray-600">{label}</span>
                    <span className="font-medium">{amount.toFixed(2)} EUR</span>
                  </div>
                  <div className="h-2 bg-gray-100 rounded-full overflow-hidden">
                    <div
                      className="h-full rounded-full bg-blue-500"
                      style={{ width: `${(amount / max) * 100}%` }}
                    />
                  </div>
                </div>
              )
            })}
          </div>
        )}
        <p className="text-sm text-gray-500 mt-4">
          Total : {finances.reduce((sum, f) => sum + f.amount, 0).toFixed(2)} EUR
        </p>
      </CardContent>
    </Card>
  )
}
```
